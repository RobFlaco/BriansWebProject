Projects

Music Player

This is one of my first projects in web development. It is a fully functional music player built with HTML, CSS, and JavaScript. It features three albums, each in various lengths, with the ability to play, pause, skip forward, and skip backward. It also features an interactive progress bar that updates as the song plays, backround color that updates based on the album, and a responsive design that looks good on both desktop and mobile devices.
